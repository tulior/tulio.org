winget.tulio.org is a fast, static package directory for Windows Package Manager.

It turns data from `microsoft/winget-pkgs` into lightweight package pages with exact package identifiers, copy-ready `winget install`, `winget upgrade`, and `winget uninstall` commands, package metadata, and version history.

The repository is `wingetdb`; the public product surface is the custom domain.

## What It Proves

- A useful package index does not need a heavy app shell.
- Static HTML can still handle search, copy flows, metadata, and crawlable package pages.
- Operationally boring tools are often the best kind: generated by a small Go binary, deployed to GitHub Pages, and built without a runtime backend.

## Design Bias

The command is the product. Everything else exists to help someone find the exact package and trust the install command before they paste it into a terminal.
